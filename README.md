# EditMantra Coding Platform

**EditMantra** is a collaborative coding platform designed to enhance computer science education. It combines real-time code editing, gamification, and learning resources to provide an interactive coding experience for students and educators.

---

## 🚀 Features

- **Real-Time Collaboration**: Multiple users can collaborate in a shared code editor with live updates.
- **Gamification**: Earn points, complete challenges, and unlock achievements to stay motivated.
- **Online Code Compiler**: Supports multiple programming languages for real-time code execution.
- **Coding Challenges**: Solve curated challenges to improve your coding skills.
- **User Authentication**: Secure login system for students and educators.

---

## 🛠️ Technology Stack

**Frontend**:
- React.js
- Monaco Editor for code editing

**Backend**:
- Node.js with Express.js
- MongoDB for data storage
- Socket.IO for real-time communication
- Judge0 for code evaluation

---

### **Prerequisites**
- Node.js (v14+)
- MongoDB installed locally or a cloud instance (e.g., MongoDB Atlas)
- Git for version control
